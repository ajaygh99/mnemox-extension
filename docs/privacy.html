Privacy Policy

Mnemox Extension · Last updated: July 14, 2026

By default, Mnemox does not collect, transmit, or store any personal data on external servers. All prompt and response scoring is processed entirely within your browser using local JavaScript.

Opt-in features

Two features are opt-in and off by default, changeable at any time from the extension popup:

  • Trace Logging — sends your prompt/response text (truncated) plus your scores to the Mnemox backend so the Traces dashboard can show your history.
  • Memory Alignment — sends AI response text to the same backend to compare it against memories you've saved.

Neither feature is active until you turn it on.

Local storage

The extension uses chrome.storage.local to remember your last prompt score, your feature-toggle settings, and an anonymous device ID between page loads. This data stays on your device unless one of the two opt-in features above is enabled.

Permissions

  • storage — save your last prompt/response score, feature-flag settings, and an anonymous device ID so the popup can display them
  • activeTab — access the current tab's text input fields
  • tabs — open the MnemoxTrace dashboard in a new tab from the popup
  • host_permissions (specific AI tool domains + the Mnemox backend) — required to run the scorer on supported AI chat sites, and to reach the backend only when Trace Logging or Memory Alignment is turned on

Data sharing and selling

Mnemox does not sell user data, does not share user data with third parties, and does not use user data for purposes unrelated to the extension's single purpose of prompt scoring and coaching.
